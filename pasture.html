Pasture

Welcome to Pasture by Headless Horse, a single-page site for resources on sustainable design, development and business.

Pasture is a Notion page rendered using Potion and hosted with clean energy. To minimise energy consumption, we have used little JS and no fonts. We have an emoji favicon too that saves us a few bytes! The only CSS used is to make colours dark, and for a notice if you’re are not using dark mode.
