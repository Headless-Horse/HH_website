Pasture

Welcome to Pasture by Headless Horse, an open-source single-page website dedicated to sustainable design and development. We've curated resources, tools, and other interesting reading materials related to sustainability.

This website is a Notion page rendered using Potion and hosted with clean energy. To minimize energy consumption, we have reduced the page size by using minimal JavaScript and no fonts. We have an emoji favicon too that helps reduce the page size. The only CSS is used to make the page colours dark, and a notice encouraging users to enable dark mode if they’re using light mode.
